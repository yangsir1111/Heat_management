本地开发环境设置

按照以下步骤设置和测试本地开发环境：

1

安装依赖

在项目根目录运行以下命令安装所有依赖：

npm install
2

启动开发服务器

在终端中运行以下命令启动前端开发服务器：

npm run dev

这将在 http://localhost:5173 启动前端应用。

3

启动后端服务器

在另一个终端中运行以下命令启动后端服务器：

npm run server

这将在 http://localhost:3001 启动后端API服务。

4

同时启动前后端

您也可以使用以下命令同时启动前后端：

npm run dev:full

这将使用 concurrently 同时运行前端和后端服务。
